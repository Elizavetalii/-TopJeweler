EShop Test Plan — как открыть и использовать

Где файл
- План тестирования: `docs/EShop_Test_Pлан.xml` (Excel‑совместимый формат SpreadsheetML).
- Отдельный файл‑таблица по аутентификации в стиле вашего примера: `docs/LumiereSecrete_TC_Authentication.xml`.

Как открыть в Excel
- macOS: двойной клик по файлу или через Finder → правой кнопкой → Open With → Microsoft Excel.
- Windows: двойной клик или Excel → File → Open → выбрать нужный `.xml` (тип "All Files").
- Альтернатива: импорт в Google Sheets (File → Import → Upload).

Как сохранить в .xlsx
- Откройте любой из файлов (`EShop_Test_Pлан.xml` или `LumiereSecrete_TC_Authentication.xml`) в Excel → File → Save As → формат `Excel Workbook (*.xlsx)`.

Что внутри
- Лист «Общая информация» — реквизиты работы.
- Лист «Описание полей» — ключевые поля/датировки.
- Лист «Test Cases» — шаблон и примеры кейсов (колонки: ID, Приоритет, Название, Предпосылки, Данные, Шаги, Ожидаемый результат, Фактический результат, Постусловия, Статус, Комментарии).
- Лист «Тестовые данные» — минимальный набор: Users, Categories, Products, ProductVariants, PromoCodes.
- Отдельный файл «TC_USR_AUT» — таблица по аутентификации c колонками и оформлением как в вашем примере.

Подсказки
- Значения в листе «Тестовые данные» можно копировать в вашу СУБД/фикстуры.
- Для расширения списка кейсов — добавляйте строки на листе «Test Cases»; Excel сохранит их при экспорте в .xlsx.
